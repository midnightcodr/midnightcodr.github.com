The idea is to

  1. group weekly data first (line #13-16), if the original problem is to do N==1 data grouping, the aggregation pipeline can stop in this step
  2. get the number of weeks (floating point) since 1970-01-01 (all OHLC data in the db are later then that date), line #21-31
  3. get the rounded number of weeks for step 2, line #37-45
  4. group by 4 weeksâ€™ OHLC, line #51-59, substitute 4 with the actual unit to suit your need
  5. sort by _id, which consists of the stock symbol and the calculated grouping number grp_weeknbr

Step 1 is essential since we need to make sure the agrregated data lands on the last trading date of the week.
